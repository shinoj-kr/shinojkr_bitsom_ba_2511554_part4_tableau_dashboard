# Task 6: Apply Visualization Design Principles

# Chart Selection Justification

## Overview

The Executive Sales Performance Dashboard is designed to provide business stakeholders with a concise overview of sales performance, profitability, customer behavior, shipping efficiency, and return analysis. The dashboard follows visualization best practices by using appropriate chart types, maintaining a clear visual hierarchy, minimizing clutter, and providing interactive filtering for business exploration.

---

# 1. Correct Chart Selection

Different chart types were selected based on the type of business question being answered.

| Visualization | Chart Type | Justification |
|---------------|------------|---------------|
| Regional Performance | Bar Chart | Ideal for comparing sales across regions because bar charts make category comparisons easy. |
| Monthly Sales Trend | Line Chart | Best suited for showing sales trends and seasonality over time. |
| Category Profitability | Horizontal Bar Chart | Allows easy comparison of profit across categories and sub-categories, especially with long category names. |
| Customer Segment View | Vertical Bar Chart | Clearly compares total sales generated by different customer segments. |
| Shipping Performance | Stacked Bar Chart | Effectively compares on-time and delayed shipments within each shipping mode. |
| Discount vs Profit | Scatter Plot | Shows the relationship between discount and profit while identifying possible outliers. |
| Return Rate by Category | Horizontal Bar Chart | Makes it easy to compare return percentages across product categories. |
| KPI Cards | Text KPIs | Highlight the most important business metrics (Sales, Profit, Orders) at the top of the dashboard. |

---

# 2. Clear Visual Hierarchy

The dashboard follows a logical information flow:

1. Dashboard title provides context.
2. KPI cards display the most important business metrics.
3. Interactive filters allow users to customize the analysis.
4. Trend and regional performance charts provide overall business performance.
5. Detailed charts provide category, customer, shipping, discount, and return insights.

This layout enables users to understand high-level performance before exploring detailed information.

---

# 3. Minimal Clutter

To improve readability:

- Unnecessary grid lines were removed.
- Only important data labels are displayed.
- Consistent spacing was maintained between charts.
- White background was used to reduce distractions.
- Only essential axis titles and labels were retained.

---

# 4. Consistent Color Usage

A consistent blue color palette was used throughout the dashboard to represent business performance.

- Darker blue highlights higher values.
- Lighter blue represents comparatively lower values.
- Pink/red is used only to indicate delayed shipments, creating clear visual contrast.

This consistency improves readability and avoids unnecessary cognitive load.

---

# 5. Proper Labels

All visualizations include:

- Descriptive chart titles
- Clearly formatted currency values (Millions)
- Percentage formatting where applicable
- KPI labels identifying each business metric
- Category names and customer segment labels for easy interpretation

---

# 6. Readable Titles

Each visualization uses a meaningful business-oriented title, including:

- Executive Sales Performance Dashboard
- Regional Performance View
- Monthly Sales Trend (2024–2025)
- Category Profitability View
- Customer Segment View
- Shipping Performance View
- Discount vs Profit
- Return Rate by Category

These titles clearly communicate the purpose of each chart.

---

# 7. Appropriate Sorting

Charts were sorted to improve comparison and insight generation.

- Regional sales are presented for easy comparison.
- Category profitability is organized to highlight higher-performing products.
- Return rate comparison clearly identifies categories with higher return percentages.

Sorting improves decision-making by making important values immediately visible.

---

# 8. Useful Filters

Three interactive filters were included:

- Region
- Category
- Customer Segment

These filters allow business users to quickly analyze specific portions of the data without creating separate reports.

---

# 9. Avoidance of Misleading Scales

The dashboard avoids misleading visualizations by:

- Using consistent axis scales.
- Maintaining proportional bar lengths.
- Using an appropriate time scale for monthly trends.
- Displaying percentages and currency values using suitable formatting.

This ensures that comparisons remain accurate and trustworthy.

---

# 10. Focus on Business Interpretation

The dashboard supports several important business decisions, including:

- Identifying the highest-performing sales region.
- Monitoring monthly sales trends.
- Comparing profitability across product categories.
- Evaluating customer segment contribution.
- Assessing shipping performance across shipping modes.
- Understanding the impact of discounts on profit.
- Monitoring return rates by category.
- Tracking overall sales, profit, and order volume using KPI cards.

Interactive filters and regional selection actions enable users to perform deeper analysis for informed business decision-making.

---

# Conclusion

The dashboard follows fundamental visualization design principles by selecting appropriate chart types, maintaining a clean and consistent layout, providing meaningful interactions, and presenting business insights in an easily understandable format. The design supports executive decision-making through clear visual communication and interactive exploration.
